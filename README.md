Tem Code Dictionary

This is Myanmar Dictionary developed for running on Desktop using JavaFX.
Dictionary Data is getting from https://github.com/saturngod/ornagai-v3/ which is MIT License.

The app will run on every deskops if installed JDK_FX (JDK with JavaFX).
If you don't have any JDK or only JDK without JavaFX, you can install JDK_FX here.
https://www.azul.com/downloads/#zulu

Opening the app will check .db file here: /ProgramData/TemCodeDictionary/mm_dictionary.db.
If not exists, it will create one for you.
